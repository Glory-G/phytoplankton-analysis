# 2026.04 浮游植物丰度与优势种分析

## 1. 项目目的

本项目分析 2026 年 4 月浮游植物样品数据。

分析分为两部分：

1.  全部样品分析
2.  B 层样品分析

全部样品用于获得整体群落结构；B 层样品从全部样品中筛选后重新计算优势种。

------------------------------------------------------------------------

## 2. 分析流程

    原始 Excel
    ↓
    数据清洗
    ↓
    计算单条记录物种丰度
    ↓
    合并同一样品中的重复物种
    ↓
    样品-物种丰度表

    ├── 全部样品分析
    │
    └── B层样品分析

------------------------------------------------------------------------

## 3. 物种丰度计算

每条记录的丰度计算：

``` math
A_{ij}=\frac{C_{ij}\times V_c}{V_m\times V_s}
```

其中：

-   $A_{ij}$：物种 i 在样品 j 中的丰度（cells/L）
-   $C_{ij}$：个体数量
-   $V_c$：浓缩体积
-   $V_m$：计数体积
-   $V_s$：采样体积

------------------------------------------------------------------------

## 4. 重复物种合并

同一样品中的同一物种按照：

-   样品编号
-   中文名
-   拉丁名
-   类群

进行合并。

合并丰度：

``` math
A_{ij}^{sum}=\sum A_{ijk}
```

------------------------------------------------------------------------

## 5. 三大类群丰度

硅藻：

``` math
A_{diatom}=\sum A_i
```

甲藻：

``` math
A_{dinoflagellate}=\sum A_i
```

蓝藻：

``` math
A_{cyanobacteria}=\sum A_i
```

总丰度：

``` math
A_{total}=\sum A_i
```

------------------------------------------------------------------------

## 6. 类群相对丰度

``` math
P_i=\frac{A_i}{A_{total}}
```

表示该类群占样品总浮游植物丰度的比例。

------------------------------------------------------------------------

## 7. 优势种计算

优势度指数：

``` math
Y_i=\frac{n_i}{N}\times f_i
```

其中：

-   $n_i$：物种累计丰度
-   $N$：全部物种累计丰度
-   $f_i$：出现频率

出现频率：

``` math
f_i=\frac{S_i}{S}
```

优势种判断：

``` math
Y_i>0.02
```

------------------------------------------------------------------------

## 8. 全部样品分析

使用全部 2026.04 样品：

计算：

-   物种丰度
-   三大类群丰度
-   类群相对丰度
-   优势种

------------------------------------------------------------------------

## 9. B层分析

筛选：

    样品编号.endswith(" B")

得到 B 层样品。

重新计算：

-   B层物种累计丰度
-   B层出现频率
-   B层优势度
-   B层优势种

不能直接从全部样品优势种中筛选，因为样品数量和频率都会变化。
