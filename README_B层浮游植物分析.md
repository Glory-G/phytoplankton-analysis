## 3. 物种丰度计算

原始数据中每一行表示某个物种在某个样品中的计数结果。

物种丰度按下式换算：

```math
A_{ij}
=
\frac{C_{ij}\times V_c}
{V_m\times V_s}
```

其中：

- \(A_{ij}\)：物种 \(i\) 在样品 \(j\) 中的丰度，单位为 cells/L；
- \(C_{ij}\)：计数得到的个体数量；
- \(V_c\)：浓缩体积，单位为 mL；
- \(V_m\)：计数体积，单位为 mL；
- \(V_s\)：采样体积，单位为 L。

对应原始表格字段：

```math
\text{物种丰度}
=
\frac{
\text{个体数量}\times\text{浓缩体积}
}{
\text{计数体积}\times\text{采样体积}
}
```

多数样品的浓缩体积为 40 mL、采样体积为 2 L、计数体积为 1 mL，因此多数记录相当于：

```math
A_{ij}=20C_{ij}
```

## 4. 同一样品内重复物种的处理

如果同一物种在同一样品中出现多条记录，则按照以下字段合并：

- 样品编号；
- 中文名；
- 拉丁名；
- 类群。

合并后的丰度为这些重复记录丰度之和：

```math
A_{ij}^{\mathrm{合并}}
=
\sum_{k=1}^{m} A_{ijk}
```

这样可以避免同一物种在同一样品中被重复统计。

## 5. 三大类群丰度计算

对每个 B 层样品，分别加总所有硅藻、甲藻和蓝藻物种的丰度。

硅藻丰度：

```math
A_{\text{硅藻},j}
=
\sum_{i\in\text{硅藻}}A_{ij}
```

甲藻丰度：

```math
A_{\text{甲藻},j}
=
\sum_{i\in\text{甲藻}}A_{ij}
```

蓝藻丰度：

```math
A_{\text{蓝藻},j}
=
\sum_{i\in\text{蓝藻}}A_{ij}
```

每个样品的浮游植物总丰度为全部类群、全部物种丰度之和：

```math
A_{\text{总},j}
=
\sum_i A_{ij}
```

---

## 6. 三大类群相对丰度计算

某类群的相对丰度等于该类群丰度除以同一样品的浮游植物总丰度。

硅藻相对丰度：

```math
P_{\text{硅藻},j}
=
\frac{A_{\text{硅藻},j}}{A_{\text{总},j}}
```

甲藻相对丰度：

```math
P_{\text{甲藻},j}
=
\frac{A_{\text{甲藻},j}}{A_{\text{总},j}}
```

蓝藻相对丰度：

```math
P_{\text{蓝藻},j}
=
\frac{A_{\text{蓝藻},j}}{A_{\text{总},j}}
```

以百分比表示时，在以上结果基础上乘以 100%。

## 7. B 层优势种计算

筛选 B 层样品后，优势种必须重新计算，不能直接从全部样品的优势种表中筛选。

### 7.1 物种累计丰度

物种 \(i\) 在全部 B 层样品中的累计丰度为：

```math
n_i
=
\sum_{j=1}^{S}A_{ij}
```

其中 \(S=17\)。

### 7.2 B 层全部物种累计丰度

```math
N
=
\sum_i n_i
```

本次结果为：

```math
N=18\,260
```

### 7.3 出现频率

物种出现频率为该物种出现的 B 层样品数占全部 B 层样品数的比例：

```math
f_i
=
\frac{S_i}{S}
```

其中：

- \(S_i\)：物种 \(i\) 出现的 B 层样品数；
- \(S=17\)：B 层样品总数。

### 7.4 总体相对丰度

```math
p_i
=
\frac{n_i}{N}
```

### 7.5 优势度指数

```math
Y_i
=
\left(\frac{n_i}{N}\right)\times f_i
```

判定标准为：

```math
Y_i>0.02
```

满足该条件的物种判定为 B 层优势种。

## 8. 计算一致性检查

本次计算通过以下检查：

| 检查项 | 结果 |
|---|---:|
| B 层样品数 | 17 |
| B 层累计总丰度 | 18,260 cells/L |
| 所有物种总体相对丰度之和 | 1.0 |
| 优势种判定阈值 | \(Y>0.02\) |
| B 层优势种数量 | 5 |

总体相对丰度之和等于 1.0，说明物种累计丰度与总丰度之间的计算关系一致。

## 10. 结果解释

菱形藻属的优势度最高，是 B 层最主要的优势种。其累计丰度为 5080 cells/L，在 17 个 B 层样品中的 16 个样品中出现，说明它同时具有较高的丰度和广泛的空间分布。

针杆藻属在全部 17 个 B 层样品中均有出现，因此其出现频率为 1.0。虽然其累计丰度略低于舟形藻属，但更高的出现频率使其优势度略高于舟形藻属。

标志星杆藻和小环藻属的累计丰度低于前三种，但由于在较多 B 层样品中出现，其优势度仍超过 0.02，因此也被判定为优势种。

本次 B 层优势种全部属于硅藻，表明硅藻是 B 层浮游植物群落的主要优势类群。甲藻和蓝藻物种在 B 层中未同时达到较高的累计相对丰度和出现频率，因此没有物种达到优势种判定阈值。
